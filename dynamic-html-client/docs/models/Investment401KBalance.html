Investment401KBalance

  • CASHBAL : String
    Cash balance available for the 401(k) account.
  • PRETAX : String
    Current value of all securities purchased with Before Tax Employee contributions.
  • AFTERTAX : String
    Current value of all securities purchased with After Tax Employee contributions.
  • MATCH : String
    Current value of all securities purchased with Employer Match contributions.
  • PROFITSHARING : String
    Current value of all securities purchased with Employer Profit Sharing contributions.
  • ROLLOVER : String
    Current value of all securities purchased with Rollover contributions.
  • OTHERVEST : String
    Current value of all securities purchased with Other (vesting) Employer contributions.
  • OTHERNONVEST : String
    Current value of all securities purchased with Other (non-vesting) Employer contributions.
  • TOTAL : String
    Current value of all securities purchased with all contributions.
  • BALLIST : BalanceList
    Beginning of generic balance list (at most one).
