ProfileResponse

  • brokerid : String
  • MSGSETLIST : MessageSetList
    List of message set information.
  • SIGNONINFOLIST : SignonInfoList
    List of signon information.
  • DTPROFUP : String
    Time this was updated on server.
  • FINAME : String
    Name of institution (FI).
  • ADDR1 : String
    FI address, line 1.
  • ADDR2 : String
    FI address, line 2.
  • ADDR3 : String
    FI address, line 3.
  • CITY : String
    FI address city.
  • STATE : String
    FI address state.
  • POSTALCODE : String
    FI address postal code.
  • COUNTRY : String
    FI address country; 3-letter country code from ISO/DIS-3166.
  • CSPHONE : String
    Customer service telephone number.
  • TSPHONE : String
    Technical support telephone number.
  • FAXPHONE : String
    Fax number.
  • URL : String
    URL for general information about FI (not for sending data).
  • EMAIL : String
    E-mail address for FI.
  • INTU.BROKERID : String
    Broker ID.
