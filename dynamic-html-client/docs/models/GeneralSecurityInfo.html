GeneralSecurityInfo

  • genSecInfoID : Integer
  • SECID : SecurityId
    Security identifier aggregate.
  • SECNAME : String
    Full name of security.
  • TICKER : String
    Ticker symbol (at most one).
  • FIID : String
    FI ID number for this security (at most one).
  • RATING : String
    Rating.
  • UNITPRICE : String
    Current price of security.
  • DTASOF : String
    Date as of for the unit price.
  • CURRENCY : Currency
    Overriding currency aggregate for unit price.
  • MEMO : String
    Memo.
