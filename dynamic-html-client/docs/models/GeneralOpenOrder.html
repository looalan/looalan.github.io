GeneralOpenOrder

  • FITID : String
    Unique FI-assigned transaction ID.
  • SRVRTID : String
    Unique server-assigned transaction ID.
  • SECID : SecurityId
    Security identified aggregate.
  • DTPLACED : String
    Date-time the order was placed.
  • UNITS : String
    Quantity of the security the open order is for.
  • SUBACCT : String
    Sub-account type. CASH, MARGIN, SHORT, OTHER.
    Enum:
    CASH
    MARGIN
    SHORT
    OTHER
  • DURATION : String
    How long the order is good for: DAY, GOODTILCANCEL, IMMEDIATE
    Enum:
    DAY
    GOODTILCANCEL
    IMMEDIATE
  • RESTRICTION : String
    Special restriction on the order: ALLORNONE, MINUNITS, NONE.
    Enum:
    ALLORNONE
    MINUNITS
    NONE
  • MINUNITS : String
    Minimum number of units that must be filled for the order.
  • LIMITPRICE : String
    Limit price.
  • STOPPRICE : String
    Stop price.
  • MEMO : String
    Other information about order (at most one).
  • CURRENCY : Currency
    Overriding currency aggregate.
  • INV401KSOURCE : String
    For 401(k) accounts, source of money for this order.
    Enum:
    PRETAX
    AFTERTAX
    MATCH
    PROFITSHARING
    ROLLOVER
    OTHERVEST
    OTHERNONVEST
