Payee

  • NAME : String
    Name of payee.
  • ADDR1 : String
    Payee’s address line 1.
  • ADDR2 : String
    Payee’s address line 2.
  • ADDR3 : String
    Payee’s address line 3. Use of <ADDR3> requires the presence of <ADDR2>.
  • CITY : String
    Payee’s city.
  • STATE : String
    Payee’s state.
  • POSTALCODE : String
    Payee’s postal code.
  • COUNTRY : String
    Payee’s country; 3-letter country code from ISO/DIS-3166.
  • PHONE : String
    Payee’s telephone number.
