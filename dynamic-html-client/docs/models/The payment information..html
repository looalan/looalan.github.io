The payment information.

  • BANKACCTFROM : BankAccount
    Account-from aggregate.
  • TRNAMT : String
    Payment amount.
  • PAYEEID : String
    Server payee identifier (required if assigned). Either <PAYEEID> or <PAYEE> can be sent, but not both.
  • PAYEE : Payee
    Complete payee billing information. Either <PAYEEID> or <PAYEE> can be sent, but not both.
  • PAYEELSTID : String
    Payee list ID (required if assigned).
  • BANKACCTTO : BankAccount
    Destination account information for systems that pay by transfers (<PAYEE> also required).
  • EXTDPMT : List
    Zero or more extended Payment aggregates.
  • PAYACCT : String
    User account number with the payee.
  • DTDUE : String
    Payment due date or the date by which payment must be received by payee.
  • MEMO : String
    Memo from user to payee.
  • BILLREFINFO : String
    Biller-supplied reference information when paying a bill, if available.
  • BILLPUBINFO : BillPubInfo
    Bill publisher information aggregate for the payment.
