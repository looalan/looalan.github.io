SignonRealm

  • fidirid : Long
    Financial institution directory identification.
  • intuitBid : Long
    Financial institution business identification.
  • characterType : String
    Type of characters allowed in password.
  • signonRealmName : String
    Name of sign on.
  • minPasswordLength : Long
    Minimum number of password characters.
  • maxPasswordLength : Long
    Maximum number of password characters.
  • caseSensitive : Boolean
    Y if password is case-sensitive.
  • specialAllowed : Boolean
    if special characters are allowed over and above those characters allowed by <CHARTYPE> and <SPACES>.
  • spacesAllowed : Boolean
    Y if spaces are allowed over and above those characters allowed by <CHARTYPE> and <SPECIAL>.
  • pinChangeSupported : Boolean
    Y if server supports <PINCHRQ> (PIN change requests).
  • changePinFirst : Boolean
  • passwordType : String
    Y if server requires clients to change USERPASS as part of first signon. Clients must ignore <CHGPINFIRST> if the profile indicates <PINCH>N. Note that if <MFACHALLENGEFIRST> is also Y, this pin change request should be sent immediately after the session containing MFACHALLENGE authentication.
  • mfasupport : Boolean
    True if this financial institution supports Multi-factor authentication.
