LoanInfo

  • LOANID : String
    Identifier of this loan.
  • LOANDESC : String
    Loan description.
  • INITIALLOANBAL : String
    Initial loan balance.
  • LOANSTARTDATE : String
    Start date of loan.
  • CURRENTLOANBAL : String
    Current loan principal balance.
  • DTASOF : String
    Date and time of the current loan balance.
  • LOANRATE : String
    Loan annual interest rate.
  • LOANPMTAMT : String
    Loan payment amount.
  • LOANPMTFREQ : String
    Frequency of loan repayments.
    Enum:
    WEEKLY
    BIWEEKLY
    TWICEMONTHLY
    MONTHLY
    FOURWEEKS
    BIMONTHLY
    QUARTERLY
    SEMIANNUALLY
    ANNUALLY
    OTHER
  • LOANPMTSINITIAL : String
    Initial number of loan payments.
  • LOANPMTSREMAINING : String
    Remaining number of loan payments.
  • LOANMATURITYDATE : String
    Expected loan end date.
  • LOANTOTALPROJINTEREST : String
    Total projected interest to be paid on this loan.
  • LOANINTERESTTODATE : String
    Total interested paid to date on this loan.
  • LOANNEXTPMTDATE : String
    Next payment due date.
