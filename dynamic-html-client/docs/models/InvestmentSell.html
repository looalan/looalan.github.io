InvestmentSell

  • INVTRAN : InvestmentTransaction
    Investment transaction aggregate.
  • SECID : SecurityId
    Security identifier.
  • UNITS : String
    For security-based actions other than stock splits.
  • UNITPRICE : String
    Price per commonly-quoted unit. Does not include markup/markdown.
  • MARKDOWN : String
    Portion of the unit price that is attributed to the dealer markdown.
  • COMMISSION : String
    Transaction commission.
  • TAXES : String
    Taxes on the trade.
  • FEES : String
    Fees applied to trade.
  • LOAD : String
    Load on the transaction.
  • WITHHOLDING : String
    Federal Tax withholdings.
  • TAXEXEMPT : String
    Tax-exempt transaction.
    Enum:
    Y
    N
  • TOTAL : String
    Transaction total.
  • GAIN : String
    For sales, total gain.
  • CURRENCY : Currency
    Overriding currency aggregate for unit price.
  • ORIGCURRENCY : Currency
    Details of the foreign currency.
  • SUBACCTSEC : String
    Sub-account type for the security: CASH, MARGIN, SHORT, OTHER.
    Enum:
    CASH
    MARGIN
    SHORT
    OTHER
  • SUBACCTFUND : String
    Where did the money for the transaction come from or go to? CASH, MARGIN, SHORT, OTHER.
    Enum:
    CASH
    MARGIN
    SHORT
    OTHER
  • LOANID : String
    For 401(k) accounts only. Indicates that the transaction was due to a loan or a loan repayment, and which loan it was.
  • STATEWITHHOLDING : String
    Used for withholdings for state taxes on a withdrawal.
  • PENALTY : String
    Indicates an amount withheld due to a penalty.
  • INV401KSOURCE : String
    For 401(k) accounts, source of money used for this security.
    Enum:
    PRETAX
    AFTERTAX
    MATCH
    PROFITSHARING
    ROLLOVER
    OTHERVEST
    OTHERNONVEST
