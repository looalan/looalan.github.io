CreditCardClosing

  • FITID : String
    Unique identifier for this statement.
  • DTOPEN : String
    Opening statement date.
  • DTCLOSE : String
    Closing statement date.
  • DTNEXT : String
    Closing date of next statement.
  • BALOPEN : String
    Opening statement balance.
  • BALCLOSE : String
    Closing statement balance.
  • DTPMTDUE : String
    Payment due date.
  • MINPMTDUE : String
    Minimum amount due.
  • FINCHG : String
    Finance charges.
  • PAYANDCREDIT : String
    Total of payments and credits.
  • PURANDADV : String
    Total of purchases and cash advances.
  • DEBADJ : String
    Debit adjustments.
  • CREDITLIMIT : String
    Current credit limit.
  • DTPOSTSTART : String
    Start date of transaction data for this statement.
  • DTPOSTEND : String
    End date of transaction data for this statement
  • MKTGINFO : String
    Marketing information.
  • IMAGEDATA : ImageData
    Image data aggregate.
  • CURRENCY : Currency
    Currency, if different from CURDEF.
  • ORIGCURRENCY : Currency
    Original currency, if different from CURDEF
