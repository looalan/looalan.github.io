Closing

  • FITID : String
    Unique identifier for this statement.
  • DTOPEN : String
    Opening statement date.
  • DTCLOSE : String
    Closing statement date.
  • DTNEXT : String
    Closing date of next statement.
  • BALOPEN : String
    Opening statement balance.
  • BALCLOSE : String
    Closing statement balance
  • BALMIN : String
    Minimum balance in statement period.
  • DEPANDCREDIT : String
    Total of deposits and credits, including interest.
  • CHKANDDEB : String
    Total of checks and debits, including fees.
  • TOTALFEES : String
    Total of all fees.
  • TOTALINT : String
    Total of all interest.
  • DTPOSTSTART : String
    Start date of transaction data for this statement.
  • DTPOSTEND : String
    End date of transaction data for this statement.
  • MKTGINFO : String
    Marketing information.
  • IMAGEDATA : ImageData
    Image data aggregate.
  • CURRENCY : Currency
    Currency, if different from CURDEF.
  • ORIGCURRENCY : Currency
    Original currency, if different from CURDEF.
