InvestmentPosition

  • SECID : SecurityId
    Security identifier.
  • HELDINACCT : String
    Sub-account type.
    Enum:
    CASH
    MARGIN
    SHORT
    OTHER
  • POSTYPE : String
    SHORT = Writer for options, Short for all others. LONG = Holder for options, Long for all others.
    Enum:
    LONG
    SHORT
  • UNITS : String
    For stocks, MFs, other, number of shares held.
  • UNITPRICE : String
    For stocks, MFs, other, price per share.
  • MKTVAL : String
    Market value of this position.
  • DTPRICEASOF : String
    Date and time of unit price and market value, and cost basis. If this date is unknown, use 19900101 as the placeholder; do not use 0.
  • CURRENCY : Currency
    Currency information if different from default currency.
  • MEMO : String
    Comment.
  • INV401KSOURCE : String
    Source of money for this security in this position.
    Enum:
    PRETAX
    AFTERTAX
    MATCH
    PROFITSHARING
    ROLLOVER
    OTHERVEST
    OTHERNONVEST
