OptionInfo

  • SECINFO : GeneralSecurityInfo
    Security information aggregate.
  • OPTTYPE : String
    Option type: PUT = put. CALL = call.
    Enum:
    CALL
    PUT
  • STRIKEPRICE : String
    Strike price.
  • DTEXPIRE : String
    Expiration date.
  • SHPERCTRCT : String
    Shares per contract.
  • SECID : SecurityId
    Security ID of the underlying security.
  • ASSETCLASS : String
    Asset Class (at most one), DOMESTICBOND, INTLBOND, LARGESTOCK SMALLSTOCK, INTLSTOCK, MONEYMRKT, OTHER.
    Enum:
    DOMESTICBOND
    INTLBOND
    LARGESTOCK
    SMALLSTOCK
    INTLSTOCK
    MONEYMRKT
    OTHER
  • FIASSETCLASS : String
    Text string containing an FI defined asset class.
