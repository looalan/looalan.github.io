Investment401K

  • EMPLOYERNAME : String
    Name of the employer.
  • PLANID : String
    Plan number.
  • PLANJOINDATE : String
    Date the employee joined the plan.
  • EMPLOYERCONTACTINFO : String
    Name of contact person at employer, plus any available contact information, such as phone number.
  • BROKERCONTACTINFO : String
    Name of contact person at broker, plus any available contact information, such as phone number.
  • DEFERPCTPRETAX : String
    Percent of employee salary deferred before tax.
  • DEFERPCTAFTERTAX : String
    Percent of employee salary deferred after tax.
  • MATCHINFO : MatchInfo
    Aggregate containing employer match information. Absent if employer does not contribute matching funds.
  • CONTRIBINFO : ContributionInfo
    Aggregate to describe how new contributions are distributed among the available securities.
  • CURRENTVESTPCT : String
    Estimated percentage of employer contributions vested as of the current date. If omitted, assume 100%.
  • VESTINFO : List
    Vest change dates. Provides the vesting percentage as of any particular past, current, or future date. 0 or more.
  • LOANINFO : List
    List of loans outstanding against this account. 0 or more.
  • INV401KSUMMARY : Investment401KSummary
    List of contributions to 401(k) account.
