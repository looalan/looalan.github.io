ExtendedPayee

  • extendedPayeeID : Integer
  • PAYEEID : String
    Server payee identifier (required if assigned). Either <PAYEEID> or <PAYEE> can be sent, but not both.
  • IDSCOPE : String
    Scope of the payee ID; one of (GLOBAL, USER).
    Enum:
    GLOBAL
    USER
  • NAME : String
    Standard payee name.
  • DAYSTOPAY : String
    Minimum number of business days needed to process.
