FiDirectory

  • fidirid : Long
    Financial directory identification.
  • intuitBid : Long
    Financial institution business identification.
  • dataId : Long
  • masterId : Long
  • name : String
    Name of the financial institution.
  • url : String
    Url address of the financial institution'w website.
  • contact : String
    Contact information of the financial institution.
  • url2 : String
    Alternate url address of the financial institution'w website.
  • directConnect : Boolean
    True if financial institution supports Direct Connect services.
  • webConnect : Boolean
    True if financial institution supports Web Connect services.
  • expWebConnect : Boolean
    True if financial institution supports Express Web Connect services.
  • active : Boolean
    True if this financial institution is active.
  • notQbp : Boolean
  • fid : String
    Financial Institution ID (unique within Org).
  • org : String
    Organization defining this FI name space.
  • profileUrl : String
    OFX profile url.
  • serviceUrl : String
    OFX service url.
  • ofxVersion : Integer
    OFX version the financial institution supports.
  • target : String
  • querystring : String
  • xmltemplate : String
  • xsltemplate : String
  • signonrealm : Boolean
    Signon realm information of the financial institution.
  • taxSupport : Boolean
    True if this financial institution supports taxation.
  • directconnect : DirectConnect
    Direct connect information.
  • signonRealm : SignonRealm
    Sign on realm information.
  • clientuid : ClientUid
    Identification for MFA support.
  • appid : String
    ID of client application.
  • appver : String
    Version of client application, (6.00 encoded as 0600).
  • preProcessor : String
  • postProcessor : String
  • rtn : String
    Alternate aggregate for bank id.
  • notes : String
    Additional notes for this financial institution.
  • csrNotes : String
    Customer service representative notes.
