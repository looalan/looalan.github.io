PresentmentNameAddress

  • NAMEACCTHELD : String
    Customer’s name as it appears on the account.
  • BUSNAMEACCTHELD : String
    Optional “Does Business As” name associated with this account.
  • ADDR1 : String
    Customer’s address line 1.
  • ADDR2 : String
    Customer’s address line 2.
  • ADDR3 : String
    Customer’s address line 3.
  • CITY : String
    Customer’s city.
  • STATE : String
    Customer’s state.
  • POSTALCODE : String
    Customer’s postal code.
  • COUNTRY : String
    Customer’s country; 3-letter country code from ISO/DIS-3166.
  • DAYPHONE : String
    Customer’s telephone number (daytime).
  • EVEPHONE : String
    Customer’s telephone number (evening).
