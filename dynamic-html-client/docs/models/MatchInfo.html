MatchInfo

  • MATCHPCT : String
    Percent of employee contribution matched.
  • MAXMATCHAMT : String
    Maximum employer contribution amount in any year.
  • MAXMATCHPCT : String
    Current maximum employer contribution percentage. Maximum match in a year is MAXMATCHPCT up to the MAXMATCHAMT, if provided.
  • STARTOFYEAR : String
    Specifies when the employer contribution max is reset. Some plans have a maximum based on the company fiscal year rather than calendar year. Assume calendar year if omitted. Only the month and day (MMDD) are used; year (YYYY) and time are ignored.
  • BASEMATCHAMT : String
    Specifies a fixed dollar amount contributed by the employer if the employee participates in the plan at all. This may be present in addition to the <MATCHPCT>. $0 if omitted.
  • BASEMATCHPCT : String
    Specifies a fixed percent of employee salary matched if the employee participates in the plan at all. This may be present in addition to the MATCHPCT>. 0% if omitted. Base match in a year is BASEMATCHPCT up to the BASEMATCHAMT,if provided.
