CreditCardAccountInfo

  • CCACCTFROM : CreditCardAccount
    Credit card account from aggregate.
  • FIBANKACCTFROM : BankAccount
    Bank account from aggregate.
  • SUPTXDL : Boolean
    Y if account supports transaction detail downloads, N if it is balance-only.
  • XFERSRC : Boolean
    Y if account is enabled as a source for an intrabank or interbank transfer.
  • XFERDEST : Boolean
    Y if account is enabled as a destination for an intrabank or interbank transfer.
  • SVCSTATUS : String
    Status of the account.
    Enum:
    AVAIL
    PEND
    ACTIVE
  • EEI.NICKNAME : String
    EEI nick name field.
  • EEI.LEDGERBAL : LedgerBalance
    EEI ledger balance field.
