ImageData

  • IMAGETYPE : String
    Defines the type of image.
    Enum:
    STATEMENT
    TRANSACTION
    TAX
  • IMAGEREF : String
    Server specified unique identifier for the image to be used during the request for the image. Can be either image identifier or URL, depending on the value of <IMAGEREFTYPE>.
  • IMAGEREFTYPE : String
    Type of reference.
    Enum:
    OPAQUE
    URL
    FORMURL
  • IMAGEDELAY : String
    Number of calendar days from <DTSERVER> (for statement images) or <DTPOSTED> (for transaction images) when image will become available.
  • DTIMAGEAVAIL : String
    Image availability date.
  • IMAGETTL : String
    Number of calendar days the image will remain available on the host once the image becomes available.
  • CHECKSUP : String
    Check image information.
    Enum:
    FRONTONLY
    BACKONLY
    FRONTANDBACK
