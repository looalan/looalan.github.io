DebtInfo

  • parvalue : String
    Par value.
  • debttype : String
    Debt type (at most one).
    Enum:
    COUPON
    ZERO
  • debtclass : String
    Classification of debt. TREASURY, MUNICIPAL, CORPORATE, OTHER.
    Enum:
    TREASURY
    MUNICIPAL
    CORPORATE
    OTHER
  • couponrt : String
    Bond coupon rate for next closest call date (at most one).
  • dtcoupon : String
    Maturity date for next coupon.
  • couponfreq : String
    When coupons mature. One of the following values: MONTHLY, QUARTERLY, SEMIANNUAL, ANNUAL, or OTHER.
    Enum:
    MONTHLY
    QUARTERLY
    SEMIANNUAL
    ANNUAL
    OTHER
  • callprice : String
    Bond call price (at most one).
  • yieldtocall : String
    Yield to next call.
  • dtcall : String
    Next call date (at most one).
  • calltype : String
    Type of next call. CALL, PUT, PREFUND, MATURITY.
    Enum:
    CALL
    PUT
    PREFUND
    MATURITY
  • yieldtomat : String
    Yield to maturity.
  • dtmat : String
    Debt maturity date (at most one).
  • assetclass : String
    Asset Class (at most one), DOMESTICBOND, INTLBOND, LARGESTOCK SMALLSTOCK, INTLSTOCK, MONEYMRKT, OTHER.
    Enum:
    DOMESTICBOND
    INTLBOND
    LARGESTOCK
    SMALLSTOCK
    INTLSTOCK
    MONEYMRKT
    OTHER
  • fiassetclass : String
    Text string containing an FI defined asset class.
  • SECINFO : GeneralSecurityInfo
    Security information aggregate.
