InvestmentTransaction

  • FITID : String
    Unique FI-assigned transaction ID. This ID is used to detect duplicate downloads.
  • SRVRTID : String
    Server assigned transaction ID.
  • DTTRADE : String
    Trade date; for stock splits, day of record.
  • DTSETTLE : String
    Settlement date; for stock splits, execution date.
  • REVERSALFITID : String
    For a reversal transaction, the FITID of the transaction that is being reversed.
  • MEMO : String
    Other information about transaction (at most one).
