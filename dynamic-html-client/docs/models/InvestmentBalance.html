InvestmentBalance

  • AVAILCASH : String
    Cash balance across all sub-accounts. Should include sweep funds.
  • MARGINBALANCE : String
    Margin balance. A positive balance indicates a positive cash balance, while a negative balance indicates the customer has borrowed funds.
  • SHORTBALANCE : String
    Market value of all short positions.
  • BUYPOWER : String
    Buying power.
  • BALLIST : BalanceList
    Beginning of Investment balance list (at most one)
