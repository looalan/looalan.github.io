Mail

  • USERID : String
    User identification.
  • DTCREATED : String
    When message was created.
  • FROM : String
    Who the message is from.
  • TO : String
    Who the message should be delivered to.
  • SUBJECT : String
    Subject of message (plain text, not HTML).
  • MSGBODY : String
    Body of message, HTML-encoded or plain text depending on <USEHTML>. HTML-encoded text - A-10000. Plain text - A-2000.
  • INCIMAGES : Boolean
    Include images in the message body.
  • USEHTML : Boolean
    Y for HTML-formatted text. N for plain text.
