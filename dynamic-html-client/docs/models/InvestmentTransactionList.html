InvestmentTransactionList

  • DTSTART : String
    Start date for transaction data.
  • DTEND : String
    This is the value that should be sent in the next <DTSTART> request to insure that no transactions are missed.
  • INVBANKTRAN : List
    Banking-related transactions for the investment account (zero or more).
  • BUYDEBT : List
    Buy debt security.
  • BUYMF : List
    Buy mutual fund.
  • BUYOPT : List
    Buy option.
  • BUYOTHER : List
    Buy other security type.
  • BUYSTOCK : List
    Buy Stock.
  • CLOSUREOPT : List
    Close a position for an option.
  • INCOME : List
    Investment income is realized as cash into the investment account.
  • INVEXPENSE : List
    Misc. investment expense that is associated with a specific security.
  • JRNLFUND : List
    Journaling cash holdings between sub-accounts within the same investment account.
  • JRNLSEC : List
    Journaling security holdings between sub-accounts within the same investment account.
  • MARGININTEREST : List
    Margin interest expense.
  • REINVEST : List
    Reinvestment of income.
  • RETOFCAP : List
    Return of capital.
  • SELLDEBT : List
    Sell debt security. Used when debt is sold, called, or reached maturity.
  • SELLMF : List
    Sell mutual fund.
  • SELLOPT : List
    Sell option.
  • SELLOTHER : List
    Sell other type of security.
  • SELLSTOCK : List
    Sell stock
  • SPLIT : List
    Stock or Mutual Fund Split.
  • TRANSFER : List
    Transfer holdings in and out of the investment account.
